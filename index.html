DevConnect

A social platform built for developers to find each other, chat in real time, and share job leads — the kind of community feature most dev tools bolt on as an afterthought. The harder part was getting presence and typing indicators to feel instant over Socket.io without flooding the server with events.

  • MERN-based developer social platform with JWT authentication and REST APIs
  • Real-time messaging and online presence tracking using Socket.io
  • State management with Zustand for efficient client-side data handling
  • Deployed on Vercel (frontend) and Render (backend) for production-grade hosting
MongoDB Express.js React.js Node.js Socket.io JWT Zustand Vercel Render

Expense Tracker

Most expense trackers just show you a list of numbers. This one summarizes your spending in plain language using the Groq API, so you get an actual answer to "where did my money go this month?" instead of a spreadsheet to interpret yourself.

  • MERN-based expense tracker with secure authentication and CRUD operations
  • AI-generated spending insights via Groq API for personalized financial analysis
  • JWT-based session management for secure user data protection
  • Deployed on Vercel with clean, responsive dashboard UI
